# Books Community API

A production-ready Laravel API for building digital book community applications. Provides a complete backend for frontend developers to create their own book sharing platforms.

## Live API Server
https://library-test-8q62.onrender.com/

## Key Features
- User Authentication (Register/Login/Logout) with Laravel Sanctum
- Complete Book Management (CRUD operations)
- Book Categories & Organization
- User Comments & Discussions
- Book Views Analytics
- Save/Unsave Books (Bookmarks)
- Book File Upload & Download
- Advanced Search & Filtering (Popular, Recent, Search)
- RESTful API endpoints
- API Documentation with Scramble

## Tech Stack
- Laravel 11.x
- PHP 8.2+
- Laravel Sanctum (API Authentication)
- Scramble (Auto-generated API Documentation)
- REST API Architecture
- Eloquent ORM

## Why Use This API?
Perfect for frontend developers looking to build book-related applications without building a backend from scratch. Ideal for learning, portfolio projects, or rapid prototyping.

## Quick Start
```bash
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```
